
A C# Windows Forms application developed during the DecodeLabs Cybersecurity Internship.

## Features
- Text encryption using Caesar Cipher
- Text decryption functionality
- Custom shift key input
- Copy encrypted/decrypted result
- Clear fields functionality
- Error handling for invalid inputs

## Technologies Used
- C#
- Windows Forms
- Visual Studio 2022
- ASCII conversion
- Modular arithmetic

## Purpose
This application demonstrates basic encryption and decryption concepts using Caesar Cipher and helps users understand cybersecurity fundamentals related to data confidentiality.
